The WidgetPack comment system replaces default WordPress comments with Real-time and Social conversations service to drive more traffic for your site.

== Description ==

WidgetPack is a Real-time comments service for makes commenting easier and help to boost your search traffic.
More information [https://widgetpack.com/comment-system](https://widgetpack.com/comment-system)

= WidgetPack for WordPress =

* It's free!
* All comments are stored in default WordPress comment system
* Comments indexable by all search engines (SEO)
* Single sign-on (SSO) and custom SSO login button
* Support schema.org/Comment microformat for fast indexable
* Import/Export of existing comments to WidgetPack
* Auto-sync of comments with WidgetPack and WordPress database
* Comment count

= WidgetPack Features =

* Star Rating
* Google Rich Snippets for Rating
* Single sign-on (SSO)
* Guest and Social login
* Real-time comments and replies
* Threaded comments and replies
* Voting for comments
* Email Notifications
* SoundCloud, Twitter, YouTube and Vimeo embedding
* Uploading images (gif, png, jpg)
* Real-time and powerful admin tools
* Spam protection with Google reCAPTCHA 2.0
* Moderator roles
* User ban, IP ban
* Group operations
* SSL

== Installation ==

**NOTE: It is recommended that you [backup your database](http://codex.wordpress.org/Backing_Up_Your_Database) before installing the plugin.**

1. Unpack archive to this archive to the 'wp-content/plugins/' directory inside of WordPress

  * Maintain the directory structure of the archive (all extracted files
    should exist in 'wp-content/plugins/widgetpack-comment-system/'

2. From your blog administration, click on Comments to change settings
